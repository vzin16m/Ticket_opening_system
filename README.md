# Ticket_opening_system
<hr>
<b>Project proposing a basis for a future ticket opening and management system</b>
<br>
<p>The following program allows the creation of a “Technician” and a “Ticket”, so a Ticket is directly linked to the technician who will resolve the call.</p>
<p>O seguinte programa permite a criação de "Técnico" e de um "Chamado", assim um Chamado está ligado diretamente ao técnico que resolverá o chamado.</p>
